# 🌿 Carbon Emission - منصة تتبع واستدامة الكربون

**Carbon Emission** هي منصة متكاملة صُممت خصيصاً لمساعدة المؤسسات على تتبع وإدارة أثرها البيئي بدقة متناهية. تم تطوير هذا النظام ليعكس رؤية الشركة في تقديم حلول تقنية مبتكرة تمزج بين التحليلات البيئية وقوة هندسة البرمجيات الحديثة.

## 🚀 حول شركة Carbon Emission
نحن نتخصص في بناء أنظمة إدارية متطورة تهدف إلى أتمتة تدفقات العمل المالية والتشغيلية. منصة **Carbon Emission** هي أحدث حلولنا التي تستخدم تقنيات Next.js 15 لضمان أداء استثنائي وتجربة مستخدم لا تضاهى.

## ✨ المميزات التقنية
* **لوحة التحكم الذكية**: تتبع حي للانبعاثات عبر النطاقات الثلاثة (Scopes 1, 2, 3).
* **حاسبة الانبعاثات الشاملة**: تغطي الانبعاثات المباشرة وغير المباشرة وسلسلة القيمة بالكامل.
* **تقارير ESG**: توليد تقارير استدامة احترافية مع سجل متكامل لمتابعة الإصدارات.
* **دعم ثنائي اللغة**: واجهة RTL كاملة للغة العربية مع استخدام خط Cairo لتحسين القراءة.

## 🛠️ التكنولوجيا المستخدمة (Tech Stack)
* **Framework**: Next.js 15 (App Router).
* **Styling**: Tailwind CSS 3.4 بتصميم عصري (Emerald Green).
* **Security**: نظام Middleware متقدم لحماية المسارات باستخدام auth_token.

## 🔐 الأمان وحماية البيانات
بصفتنا خبراء في أتمتة الأنظمة، تم تزويد المنصة بـ:
* حماية صارمة لجميع المسارات الحساسة مثل /dashboard و /settings.
* نظام حماية يمنع الوصول غير المصرح به ويعيد التوجيه تلقائياً لصفحة تسجيل الدخول.

# 🌿 Carbon Emission - Tracking & Sustainability Platform

**Carbon Emission** is a comprehensive platform designed specifically to help organizations track and manage their environmental impact with extreme precision. This system reflects the company's vision of providing innovative technical solutions that blend environmental analytics with modern software engineering.

## 🚀 About Carbon Emission
We specialize in building advanced management systems aimed at automating financial and operational workflows. The **Carbon Emission** platform is our latest solution using Next.js 15 technologies to ensure exceptional performance and an unmatched user experience.

## ✨ Technical Features
* **Smart Dashboard**: Real-time emissions tracking across Scopes 1, 2, and 3.
* **Comprehensive Calculator**: Covers direct, indirect, and value chain emissions.
* **ESG Reporting**: Generation of professional sustainability reports with a full version control log.
* **Bilingual Support**: Full RTL support for Arabic, utilizing the Cairo font for enhanced readability.

## 🛠️ Technology Stack
* **Framework**: Next.js 15 (App Router).
* **Styling**: Tailwind CSS 3.4 with a modern Emerald Green theme.
* **Security**: Advanced Middleware system for route protection using auth_token.

## 🔐 Security & Data Protection
As experts in system automation, the platform is equipped with:
* Strict protection for all sensitive routes such as /dashboard and /settings.
* A security system that prevents unauthorized access and automatically redirects to the login page.
